# Simple Order System

A real-world ASP.NET Core 8 web application built using Clean Architecture, MVC + Web API, SQL Server, and ASP.NET Core Identity.

## 🏗 Architecture
- Clean Architecture
- Domain-driven design principles
- Dependency Injection
- Repository Pattern
- Singleton Pattern (Order Number Generator)

## 🧩 Technologies
- ASP.NET Core 8 (MVC + API)
- Entity Framework Core
- SQL Server
- ASP.NET Core Identity
- Bootstrap

## 🔐 Security
- Authentication & Authorization
- Role-based access (Admin, User)
- Password hashing
- HTTPS enforcement

## 🎯 Features
- User registration & login
- Product listing
- Order creation
- Order history
- Admin-only access areas

## 🧠 Design Patterns Used
- Singleton
- Repository
- Dependency Injection

## 🚀 How to Run
1. Clone the repository
2. Update connection string
3. Run migrations
4. Start the application

## 📌 Learning Outcome
This project demonstrates how to structure scalable ASP.NET Core applications using professional best practices.
